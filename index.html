WhiskTube

The offline YouTube downloader!

Download .zip Download .tar.gz View on GitHub

WhiskTube is a Node.js local webapp for downloading videos from YouTube to save them later. It's quite easy to use provided you have some knowledge of terminals/node.js.

To install, download the project. You will need Node.js and NPM. Then, run npm install to install the project's dependencies. After that, npm start starts WhiskTube. You can access it via the link provided.
